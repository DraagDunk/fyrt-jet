# Fyrtøjet

Dette er et tekst-baseret rollespil baseret på H. C. Andersens "Fyrtøjet".

Det er ret janky, men det har været sjovt at lave.
 
Start: `python3 -m game`

Du kan altid skrive "g" i spillet for at se en liste over dine genstande.

Du kan altid skrive "k" i spillet for at se en oversigt over din karakter.
